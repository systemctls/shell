bin:放置二进制文件程序工具，它是系统必须的基本工具集。我用来存放常用的系统脚本，按操作系统来细分
opt:用来安装第三方软件。我用来存放第三方软件，每个软件基本配一个使用手册，附录文件官网地址，常用操作指南。
root:超级用户。我用来存放整理后的归档文件，包含公司与个人，必须附录说明文件。
tmp:放置每个用户的临时文件，这个目录是全局可写的。用来放置收集的文件，临时文件，待整理的文件。
usr:所有用户的公共通用的配置资料、工具、文档。用来存放个人日常文件，比如音乐，视频，文档
var:是variable(变量)的缩写，这个目录中存放着在不断扩充着的东西，我们习惯将那些经常被修改的目录放在这个目录下。用来存放tmp下整理后的文件或者软件缓存目录
run:是一个临时文件系统，存储系统启动以来的信息。当系统重启时，这个目录下的文件应该被删掉或清除.临时存储，随意删除
etc:放置系统所有的配置文件，我们修改系统配置主要就是这个目录下的文件。用来存放软件配置文件，达到开箱即用
usr/inbox:个人收集箱
usr/project:存放项目
usr/document:存放文档
usr/picture:存放照片哦
usr/video:存放电影
usr/music:存放音乐
usr/download:下载目录 
usr/application:存放各种软件

boot: 放置系统启动的必须文件（系统启动的必须文件、系统启动必须的内核、引导程序配置文件等）
dev: 放置系统的设备文件（外设、磁盘等）
home: 放置普通的用户文件
initrd: 临时目录，初始化引导时所用，一般是控的基本上不用
lib: linux下可执行程序共享动态链接库
lost+found: 放置系统启动是自检发现的碎片文件，一般都是空的
misc: 放置杂七杂八的额外文件，一般是空的
mnt: 放置挂载外部设备文件
proc: 虚拟的文件系统、内核进程的资料系统。不占用硬盘空间
sbin: 放置超级用户的基本管理工具
var: 放置动态变化的数据、日志等。
